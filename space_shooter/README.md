# Space Shooter 🚀

A classic arcade-style space shooter game built with Python and Tkinter.

## How to Run

```bash
cd space_shooter
python main.py
```

## Controls

| Key | Action |
|-----|--------|
| ← → or A/D | Move left/right |
| SPACE | Shoot bullets |
| ESC | Quit game |
| Click or ENTER | Start / Restart game |

## Gameplay

- Control your spaceship and defend Earth from alien invaders
- Shoot enemies to earn points (100 points per enemy)
- Avoid enemy collisions - you have 3 lives
- After losing all lives, press ENTER or click to restart

## Features

- Smooth scrolling star field background
- Explosion particle effects
- Invulnerability frames (blinking) after being hit
- Score tracking and lives display
- Progressive difficulty through increasing enemy spawns over time
